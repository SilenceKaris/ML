# UDCA应答不佳风险预测模型

## 项目说明
本仓库包含原发性胆汁性胆管炎（PBC）UDCA应答不佳风险预测模型的部署代码。

## 数据安全声明
⚠️ 本仓库仅包含预测模型及部署代码，不包含任何患者临床数据。
